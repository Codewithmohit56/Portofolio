Project 1

Online FoodShop is a responsive mini web project that simulates an online food ordering platform. It allows users to browse restaurants, explore categorized menus, view daily specials, add items to a cart, and place an order through a simple checkout flow. The project focuses on clean UI design, responsive layout, and interactive features to demonstrate practical front-end development skills.

Built With:

  • HTML5

  • CSS3

  • JavaScript (ES6)

  • Responsive Web Design
